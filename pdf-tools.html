How to merge PDF files online

  1. Click the upload box under "Merge PDF" and select two or more PDF files from your device.
  2. Check the file order in the list — files merge top to bottom.
  3. Click "Merge PDFs". Your browser combines them instantly and downloads the result — no file is sent to any server.

Frequently asked questions

Are my PDF files uploaded anywhere?

No. All merging, splitting and watermarking happens locally in your browser using JavaScript — your files never leave your device.

Is there a file size or page limit?

The only limit is your device's available memory. Most PDFs up to a few hundred MB work without issue.

Do I need to create an account?

No sign-up is required for any tool on Workbench.
